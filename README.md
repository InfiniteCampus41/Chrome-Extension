# How To Install
Step 1: Click On The Code Icon Near The Top Right
Step 2: Click Download Zip File
Step 3: Find The Zip File On Your Computer, Right Click And Then Click Extract
Step 4: Go To chrome://extensions And Switch The Developer Mode Toggle In The Upper Right To On
Step 5: Click The Load Unpacked Button And Select The Folder That Was Extracted
Step 6: Pin The Extension To Your Toolbar
Step 7: Go To Any Site Which You Wish To Have Custom CSS Or JS On
Step 8: Right Click The Extension And Click Add URL
Step 9: Right Click The Extension And Click Options
Step 10: Code Your Custom CSS And JS And Click Save
Step 11: Reload The Site Which Now Has Custom Code
